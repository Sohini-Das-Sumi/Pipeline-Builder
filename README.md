# Pipeline Builder - GitHub Repository Description

---

**Pipeline Builder** is a visual, node-based workflow construction application that enables users to create, configure, and execute data processing pipelines through an intuitive drag-and-drop interface.

**Key Features:**
- Visual Pipeline Editor with ReactFlow for intuitive workflow design
- Multiple Node Types: Input, LLM, Output, Text, Filter, Database, Image, Timer, Note, and Custom nodes
- LLM Integration supporting Ollama (local), OpenAI, and Anthropic models
- Database Connectivity with SQLite, PostgreSQL, MySQL, and MongoDB support
- Real-time Pipeline Execution with live feedback
- Dark/Light Theme Support
- 3D Animated Background using Three.js
- Automatic State Persistence to localStorage

**Technology Stack:**
- **Frontend:** React 18.2.0 with Vite, ReactFlow, Zustand-style state management, Tailwind CSS, Three.js, GSAP
- **Backend:** Python FastAPI (Port 8003), Node.js Express (Port 5001)
- **APIs:** Ollama, OpenAI, Anthropic

Perfect for building AI workflows, data processing pipelines, and automation sequences with a modern visual interface.
